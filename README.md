# 🎸 Brkovi Fan Site – Custom WordPress Theme Portfolio

Welcome to the **Brkovi Fan Site**, a professional demo project designed to showcase the power of **Custom WordPress Development**. This theme is fast, lightweight, and fully scalable, built without the use of heavy page builders like Elementor or Divi.

---

## 🚀 The Stack

This project follows a modern approach to WordPress development to ensure maximum performance and maintainability:

- **PHP 8.x** – Clean logic and backend architecture.
- **Bootstrap 5** – For a modern, mobile-first responsive design.
- **Carbon Fields** – For creating advanced Gutenberg blocks and a robust Theme Options panel.
- **LeafletJS** – Lightweight, interactive maps without the overhead of Google Maps.
- **Vanilla JavaScript** – Minimal library usage for top-tier loading speeds.

---

## ✨ Key Features

### 🧩 Custom Gutenberg Blocks
The site is built modularly. Users can build pages by dragging and dropping unique, tailor-made blocks:
- **Leaflet Map Block:** Dynamic coordinate input directly within the editor.
- **Team/Band Members Block:** Advanced block featuring skill progress bars and social media integration.
- **Merch Shop Block:** Built using `complex` fields for easy product management.

### ⚙️ Theme Options Panel
The entire website is controlled from a single centralized location. The admin panel includes a dedicated menu for:
- **General:** Change logo, favicon, and site branding.
- **Contact Info:** Global management of phone numbers, emails, and addresses.
- **Integrations:** Easily inject Google Analytics, FB Pixel, or custom scripts through a dedicated tab.

### ⚡ Speed Optimized (SEO Ready)
Unlike sites built with drag-and-drop builders, this theme achieves a **95+ Google PageSpeed Insights score**. By rendering content via PHP and optimized CSS, we eliminate "code bloat" and improve search engine rankings.

---

## 🛠️ Installation

1. Clone the repository into your `/wp-content/themes/` folder.
2. Install dependencies (if using Composer):
   ```bash
   composer install
   ```
3. Activate the theme in the WordPress Admin Dashboard.
4. Navigate to **Theme Options** to set up your global configuration.
